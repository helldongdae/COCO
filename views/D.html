Problem D. 뱅크런

한동은행이 파산을 선언, 사람들이 은행 본사 앞으로 뱅크런을 하기 위해 몰려들었다. 모인 사람들은 모두 통장에 있는 본인 잔고를 전부 인출하려고 한다. 한동은행은 본사에 남은 현금을 모두 사용하여, 본인 잔고를 모두 찾는 사람의 수가 최대가 되도록 돈을 지급하기로 결정했다. 사람들의 통장잔고와 한동은행 본사에 남은 현금의 양이 주어질 때, 본인 잔고의 돈을 모두 찾을 수 있는 사람 수의 최대값을 구하는 프로그램을 작성하라.

입력

- 입력은 input.txt로 주어진다. 입력의 첫 줄에는 테스트 케이스의 개수가 주어진다.
- 각 테스트 케이스의 첫째 줄에 한동은행 ATM에 현재 들어있는 돈 N이 주어진다.

0 < N <= 1000000

- 각 테스트 케이스의 둘째 줄에는 ATM기 앞에 서 있는 사람의 수 M 명이 주어진다.

0 < M <= 10000

- 각 테스트 케이스의 셋째 줄부터 M+2번째 줄까지 각 사람의 통장에 들어있는 돈 K가 한 줄에 하나씩 주어진다.

0 < K <= 1000000

출력

잔고를 모두 찾는 사람의 수가 최대가 되는 방향으로 돈을 나누어 줄 경우, 잔고를 모두 찾게 되는 사람의 최대 수를 각 테스트 케이스 당 한 개의 정수로 한 줄에 하나씩 output.txt에 출력하여라.

Sample Input

2
100
5
20
50
10
30
20
50
3
10
20
20

Sample Output

4
3
